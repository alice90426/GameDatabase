# Project Rules

This is a professional game portfolio website.

Design style:
- Dark futuristic UI
- Clean dashboard style
- Avoid flashy casino effects
- Professional game studio feeling

Tech stack:
- Next.js
- Tailwind CSS
- TypeScript

Code rules:
- Use reusable components
- Use clean folder structure
- Avoid huge files
- Prefer modular design

UI rules:
- Responsive
- Minimal animations
- Card-based layouts
- Consistent spacing

Data:
- Games are stored in JSON
- Support filtering
- Support multilingual content
