# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：重複得分、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。
- [WD]出現在第2、3或4滾輪。
- [WD]出現在高賠圖標5連線上，即可獲得5場免費遊戲。

### 分散圖標
- [SC]只出現在免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 免費遊戲中，滾輪上任意位置出現[SC]，即可獲得獎勵。
- 每個[SC]獎勵分數與觸發此免費遊戲的連線分數相同。
- 每多一個[SC]，該廠的[SC]重複得分將增加1倍，並再獲得1場免費遊戲。
