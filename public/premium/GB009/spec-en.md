# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Repeat Win, Free Game.
---

## Symbols
|  Symbol     | Description     |
|-------------|-----------------|
| [H1]~[H5]   | High Pay        |
| [L1]~[L5]   | Low Pay         |
| [WD]        | Wild            |
| [SC]        | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].
- [WD] only appear on reel 2, 3, 4.
- 5 free spins are won for 5 of a kind high pay symbols with one or more [WD].

### Scatter Symbol
- [SC] only appears in Free Game.
---

## Game Features

### Free Game
- During Free Game, any [SC] appears on the reel win a prize. 
- The prize of each [SC] is the same as the win line that triggered the free game.
- For each additional [SC], the [SC] repeat win increases 1X and you get 1 more free spin.
