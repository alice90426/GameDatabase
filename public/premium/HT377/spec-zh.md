# 遊戲資訊

## 簡介
- 畫面: 5 輪，3 列。
- 線數: 486 路。
- 特色：雙向賠付、消除滾輪、得分倍數、免費遊戲。
---

## 符號
|  符號        | 說明  |
|------------|-----|
| [H1]~[H4] | 高獎項 |
| [H5]~[H8] | 中獎項 |
| [L1]~[L4] | 低獎項 |
| [WD]      | 百搭  |
| [SC]      | 分散  |

### 百搭符號
- [WD]出現在第2、3或4滾輪。
- [WD]可替代除了[SC]以外的所有圖標。

### 分散圖標
- 滾輪上任意位置出現3個[SC]，即可獲得6場免費遊戲。每額外1個[SC]增加2場免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 滾輪上任意位置出現3個[SC]，可再獲得6場免費遊戲。每額外1個[SC]增加2場免費遊戲。
- 觸發免費遊戲時，主遊戲中的遞增倍數會成為免費遊戲的起始倍數。
- 免費遊戲期間，遞增倍數會保留至免費遊戲結束。

### 消除轉輪
- 所有得獎符號會消除，並在隨機位置產生 1 個 [WD]，上方符號會向下掉落形成新一輪。
- 每一輪的新得獎組合都會累計，直到沒有更多得獎組合為止。

### 遞增倍數
- 每次主遊戲旋轉開始時，左側與右側倍數皆為 x1。
- 新符號掉落後，倍數會依消除符號數量增加。
- 左側倍數：套用於由左至右的得獎組合。
- 右側倍數：套用於由右至左的得獎組合。
- 五連線組合會同時增加兩側倍數。
- 得獎金額會乘上對應方向的倍數。
- 五連線得獎會乘上所有倍數相乘後的值。
- 遞增倍數會保留至該次旋轉結束。
