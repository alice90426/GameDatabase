# 遊戲資訊

## 簡介
- 盤面：5滾輪，5列。
- 線數：50線。
- 特色：獎勵輪盤、收集、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H5] | 高賠     |
| [L1]~[L5] | 低賠     |
| [WD]      | 百搭     |
| [SC]      | 分散     |
| [BN]      | 獎勵     |
| [BW]      | 百搭獎勵 |

### 百搭圖標
- [WD] 替代除 [SC] 和 [BN] 之外的所有圖標。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將使用 8/12/15 免費局數觸發/再觸發免費遊戲。

### 獎勵圖標
- 當任意位置出現 6 個或更多 [BN]/[BW] 圖標時，將觸發獎勵遊戲和 3 個獎勵局數。
- [BW] 替代除 [SC] 和 [BN] 之外的所有圖標。
- [BN] 圖標的倍數為 0.5x、0.75x、1x、1.5x、1.75x、2x、2.5x、3x、3.5x、4x、5x、10x 或獎勵輪盤。
---

## 遊戲特色

### 獎勵輪盤
- 當 [H1]/[WD] 填充主遊戲中的所有盤面時，將觸發獎勵輪盤 1 次旋轉。
- 輪盤可能會獎勵 15x、20x、25x、30x、40x、50x、100x 和 200x。

### 免費遊戲
- 盤面中心位置始終為 [BW]。
- [BW] 圖標將收集其他 [BN] 圖標點。

### 獎勵遊戲
- [BN] 圖標將被鎖定在盤面上，並在獎勵遊戲結束後作為獎金收集。
