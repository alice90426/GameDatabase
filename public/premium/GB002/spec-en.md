# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 10 Lines.
- Features: Mixed Symbol, Bonus Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H4] | High Pay        |
| [L1]~[L3] | Low Pay         |
| [SC]      | Scatter         |

### Scatter Symbol
- 3, 4 or 5 [SC] anywhere on the reels trigger Bonus Game.
---

## Game Features

### Bonus Game
- There are 3 chances of selection when the Bonus Game trigger.
- The multiplier in the Bonus Game according to the number of [SC].
