# Game Information

## Introduction
- Size: 5 reels, 7 rows.
- Lines: 117649 Ways.
- Features: Split Symbol, Multiplier Symbol, Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H5] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [W1]      | Split       |
| [W2]      | Multiplier  |
| [SC]      | Scatter     |

### Wild Symbol
- [WD]/[W1]/[W2] appears on reels 2, 3, 4 and 5 only.
- [WD] substitutes for all symbols except [W1], [W2] and [SC].
- [W1] substitutes for all symbols except [W2] and [SC].
- [W2] substitutes for all symbols except [W1] and [SC].

### Scatter Symbol
- When 4/ 5/ 6 [SC] symbols appear anywhere will trigger the Free Game with 10/ 12/ 15 free spins.
---

## Game Features

### Split Symbol
- When [W1] appears, random number of symbols on reels 2 to 5 excluding [SC], [WD], [W1] and [W2] will split to 2.

### Multiplier Symbol
- When [W2] appears, random number of reels from reels 2 to 5 will award multiplier to all symbols on those reels.
- The multiplier is the same as that reel height.

### Free Game
- When 3 or more [SC] symbols appear anywhere will retrigger the Free Game with 5 free spins.
- Each spin will trigger Split Symbol Feature.
