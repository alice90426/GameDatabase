# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：9線。
- 特色：延展百搭、得分倍數、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L5] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |
| [FG]      | 免費         |
| [BN]      | 獎勵         |

### 百搭圖標
- [WD]可替代除了[SC]、[FG]和[BN]以外的所有圖標。
- [WD]出現在第2、3或4滾輪。
- [WD]出現滾輪會替換該滾輪其他圖標，使該滾輪全部符號皆為[WD]。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得3、10或100倍總押注得分。

### 免費圖標
- 第1、3或5滾輪上任意位置出現3個[FG]，即可獲得10場免費遊戲。

### 獎勵圖標
- 第1、2或3滾輪上任意位置出現3個[BN]，即可獲得獎勵遊戲。
---

## 遊戲特色

### 免費遊戲
- 觸發免費遊戲會額外獲得隨機加乘倍數(2倍、4倍或6倍)
- 第1、3或5滾輪上任意位置出現3個[FG]，可再獲得10場免費遊戲。

### 獎勵遊戲
- 從6個選項中選取一個，並且獲得該選項對應倍數。
