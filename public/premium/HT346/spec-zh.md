# 遊戲資訊

## 簡介
- 盤面：5滾輪，4列。
- 線數：1024 路。
- 特點：隨機獎品，雙盤面，獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |
| [BN1]     | 倍數 |

### 百搭圖標
- [WD] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC] 和 [BN] 之外的所有圖標。

### 分散圖標
- [WD] 僅出現在盤面 2、3 和 4 上。
- 當 3 [SC] 圖標出現在任意位置時，將觸發獎勵遊戲和 8 獎勵局數。

### 獎勵圖標
- 當[BN]和[BN1]出現在同一行時，玩家將獎勵[BN]上的所有獎勵並乘以[BN1]。
- [BN] 攜帶 0.5x、0.75x、1x、1.5x、2x、3x、4x、5x、10x、20x、100x 或獎勵遊戲/ 重轉的倍數。
- [BN1] 僅出現在滾輪 1 上。
- [BN1] 帶有 X1 到 X5 的倍數。
---

## 遊戲特色

### 獎勵遊戲
- 獎勵遊戲在 5x5 雙盤面上播放。 （左-盤面和右-盤面）。
- 只有[BN]和[BN1]會出現在盤面上，而[BN1]可能會被鎖定在盤面上，直到獎勵遊戲結束。
- [BN1] 出現在左側盤面的第 1 滾輪處和右側盤面的第 5 滾輪處。
- 當 [BN] 與重轉和 [BN1] 出現在同一行時，另一個滾輪將重轉一次，並且 [BN] 將被鎖定。
- 當[BN1]和4個[BN]出現在同一行時，[BN1]將被升級。
  - 如果 [BN1] 沒有被鎖定，它將被鎖定。
  - 如果 [BN1] 被鎖定並且倍數不是 X5，則倍數將增加 1。
  - 如果 [BN1] 被鎖定並且倍數為 X5，則將以 3 獎勵局數再觸發獎勵遊戲。
