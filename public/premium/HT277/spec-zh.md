# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特點：複製獲勝，免費遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H4] | 高賠     |
| [L1]~[L5] | 低賠     |
| [WD]      | 百搭     |
| [SC]      | 分散     |
| [HW]      | 超級百搭 |

### 百搭圖標
- [WD]/[HW] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC] 和 [HW] 之外的所有圖標。
- [HW] 替代除 [SC] 之外的所有圖標。

### 分散圖標
- 僅出現在免費遊戲中。
---

## 遊戲特色

### 免費遊戲
- 當獲勝連接包含[HW]時，將用8個免費局數觸發/再觸發免費遊戲，並且得分點將被記錄為複製得分。
- 當[SC]出現在免費遊戲中時，玩家將獲得複製得分。每個 [SC] 圖標獲得 1 複製得分。
