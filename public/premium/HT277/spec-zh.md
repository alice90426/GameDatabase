# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特點：複製獲勝，免費遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H4] | 高賠     |
| [L1]~[L5] | 低賠     |
| [WD]      | 百搭     |
| [SC]      | 分散     |
| [SW]      | 超級百搭 |

### 百搭圖標
- [WD]或[SW]出現在第2、3、4或5滾輪。
- [WD]可替代除了[SC]和[SW]以外的所有圖標。
- [SW]可替代除了[SC]以外的所有圖標。

### 分散圖標
- [SC]只出現在免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 當獲勝連接包含[SW]時，將用8個免費場次觸發或再觸發免費遊戲，並且得分點將被記錄為複製得分。
- 當[SC]出現在免費遊戲中時，即可獲得複製得分。每個 [SC] 圖標獲得 1 複製得分。
