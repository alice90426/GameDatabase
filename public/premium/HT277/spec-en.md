# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Repeat Win, Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H4] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [SC]      | Scatter     |
| [SW]      | Super Wild  |

### Wild Symbol
- [WD] or [SW] appears on reels 2, 3, 4 or 5.
- [WD] substitutes for all symbols except [SC] and [SW].
- [SW] substitutes for all symbols except [SC].

### Scatter Symbol
- [SC] appears in Free Game only.
---

## Game Features

### Free Game
- When a winning connection include [SW] will trigger or retrigger Free Game with 8 free spins, and the win point will be record as duplicated win.
- When [SC] appears in Free Game, player will obtain the duplicated win. Each [SC] symbol obtain 1 duplicated win.
