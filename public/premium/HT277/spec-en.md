# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Duplicated Winning, Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H4] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [SC]      | Scatter     |
| [HW]      | Super Wild  |

### Wild Symbol
- [WD]/[HW] appears on reels 2, 3, 4 and 5 only.
- [WD] substitutes for all symbols except [SC] and [HW].
- [HW] substitutes for all symbols except [SC].

### Scatter Symbol
- Only appears in Free Game.
---

## Game Features

### Free Game
- When a winning connection include [HW] will trigger/retrigger Free Game with 8 free spins, and the win point will be record as duplicated win.
- When [SC] appears in Free Game, player will obtain the duplicated win. Each [SC] symbol obtain 1 duplicated win.
