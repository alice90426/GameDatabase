# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：50線。
- 特色：巨型圖標，選擇免費遊戲，獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD] 替代除 [SC] 和 [BN] 之外的所有圖標。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將觸發免費遊戲和 4/5/6 免費局數。

### 獎勵圖標
- 當任意位置出現 6 個或更多 [BN] 圖標時，將觸發 3 個獎勵局數的獎勵遊戲。
- 當每個滾輪至少有 1 個 [BN]/[WD] 圖標時，[WD] 圖標將轉化為 [BN]，玩家將獲得其倍數。
- [BN] 圖標的倍數為 1x、2x、3x、4x、5x、6x、7x、8x、9x、10x、18x、20x 或 80x。
---

## 遊戲特色

### 免費遊戲
- 每次旋轉 2、3 和 4 滾輪將出現 3x3 巨型圖標。

### 獎勵遊戲
- 當新的 [BN] 出現在盤面上時，剩餘旋轉重置為 3。
- [BN] 圖標將鎖定在盤面上，並在獎勵遊戲結束後結算。
- 盤面中的每個位置可以具有 X2、X3 或 X5 的倍數。 [BN] 圖標將乘以倍數。
- 當所有盤面全部滿時，玩家將獲得 500x。
