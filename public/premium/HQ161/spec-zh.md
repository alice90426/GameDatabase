# 遊戲資訊

## 簡介
- 盤面：5滾輪，5列。
- 線數：88線。
- 特色：消除滾輪、百搭倍數、得分倍數、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L4]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 倍數百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]/[W1]堆疊在第3、4、5滾輪。
- [W1]只出現在免費遊戲，得分經過[W1]乘上2X。

### 分散圖標
- 3、4或5個[SC]出現在滾輪上任意位置上時，即可獲得12、16或20場免費遊戲與5、25或250倍的總押注得分。
- [SC]只出現在主遊戲。
---

## 遊戲特色

### 消除滾輪
- 任何得分都會觸發消除滾輪特色，直到沒有贏分為止。

### 免費遊戲
- 觸發消除次數越多，額外乘上的倍數越高。倍數從2倍、3倍、4倍，最高到5倍。
- 當連線經過多個[W1]時，倍數可連乘，最高8倍。
