# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Multiplier Wild, Select Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H5] | High Pay        |
| [L1]~[L6] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].
- [WD] appears on reels 2, 3 or 4.

### Scatter Symbol
- 3, 4 or 5 [SC] from leftmost to right on adjacent reels trigger Select Free Game, and win 5x, 10x or 50x total play amount.
---

## Game Features

### Select Free Game
- Five Featured Free Games can be chosen when a Free Game be triggered.
- Associated with the spin times, player can choose fewer spin times to get a higher multiplier.
- If the combination play amount includes [WD], the winning prize will be multiplied by multiplier.
- 3 or more [SC] from leftmost to right on adjacent reels retrigger Select Free Game.
- Another Free Game can be chosen when the original Free Game is ended.
