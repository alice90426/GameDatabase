# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：得分倍數、免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L5] | 低賠         |
| [WD]      | 百搭         |
| [W1]      | 升級百搭     |
| [SC]      | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只出現在第2、3、4或5滾輪。
- [W1]只出現在第3滾輪。
- [W1]出現時，可使免費遊戲起始倍數上升，最高等級5。

### 分散圖標
- 滾輪上由最左至右出現3、4或5個[SC]，即可獲得8、13或20場免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 第3滾輪出現[W1]時，可獲得1場免費遊戲並增加得分倍數，增加的倍數與免費遊戲等級相同。
