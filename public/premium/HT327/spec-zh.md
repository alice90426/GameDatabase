# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特點：三重能量、神秘圖標、百搭乘法器、收集、免費遊戲、彩金。
---

## 圖標
|  圖標       | 描述 |
|-------------|------|
| [H1]~[H5]   | 高賠 |
| [L1]~[L5]   | 低賠 |
| [WD]        | 百搭 |
| [SC1]~[SC3] | 分散 |
| [MS]        | 神秘 |

### 百搭圖標
- [WD] 替代除 [SC] 之外的所有圖標。

### 分散圖標
- 當任何 [SC1]~[SC3] 圖標出現在任何地方時，都可能觸發免費遊戲或累積不同的特色。
  - [SC1]：觸發免費遊戲並累加免費局數。
    - 初始免費局數是 9。
  - [SC2]：使用倍數 [WD] 功能觸發免費遊戲。
    - 初始倍數為 X2，最高可達 X7。
  - [SC3]：使用彩金功能觸發免費遊戲。
    - 彩金倍數為 3x、8x、15x、50x、200x 或 500x。
- [SC1]~[SC3]可以同時觸發，共有7種免費遊戲。
---

## 遊戲特色

### 神秘圖標
- 盤面上的所有 [MS] 圖標每次旋轉都會轉換為相同的圖標。

### 百搭倍數
- 獎金將乘以倍數。

### 免費遊戲
- 如果在沒有 [SC1] 的情況下觸發免費遊戲，則免費局數為 7。
- 如果免費遊戲由 [SC1] 觸發，則免費局數將是累積的免費局數。
- 在彩金功能中，所有彩金都可以多次收集。
