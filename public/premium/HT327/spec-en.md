# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 25 Lines.
- Features: Triple Power, Mystery Symbol, Multiplier Wild, Collect, Free Game, Jackpot.
---

## Symbols
|  Symbol     | Description |
|-------------|-------------|
| [H1]~[H5]   | High Pay    |
| [L1]~[L5]   | Low Pay     |
| [WD]        | Wild        |
| [SC1]~[SC3] | Scatter     |
| [MS]        | Mystery     |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- 1 or more [SC1], [SC2] or [SC3] anywhere on the reels may trigger Free Game or accumulate the different feature.
  - [SC1]: Trigger Free Game with accumulate free spins.
    - The initial free spins is 9.
  - [SC2]: Trigger Free Game with multiplier [WD] feature.
    - The initial multiplier is X2, and up to X7.
  - [SC3]: Trigger Free Game with Jackpot feature.
    - The Jackpot multiplier is 3x, 8x, 15x, 50x, 200x or 500x.
- [SC1]~[SC3] may trigger at the same time, there are 7 type Free Games.
---

## Game Features

### Mystery Symbol
- All [MS] symbols on the reels will transform to the same symbol each spin.

### Multiplier Wild
- The winning will be multiplied by the multiplier.

### Free Game
- If the Free Game is triggered without [SC1], the free spins is 7.
- If the Free Game is triggered with [SC1], the free spins will be the accumulated free spins.
- In the Jackpot feature, all Jackpot can be collected multiple times.
