# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：滿輪百搭、倍數百搭、隨機百搭、得分倍數、多選項免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H6]   | 高賠         |
| [L1]~[L6]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。
- 得分連線上有出現[WD]，該線得分將額外乘上2X。(不包含[WD]連線)

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得多選項免費遊戲與2、20或200倍總押注得分。
---

## 遊戲特色

### 滿輪百搭
- 主遊戲隨機觸發，觸發時可能隨機出現1~5個滿輪百搭。

### 多選項免費遊戲
- 兩種特色免費遊戲可選擇。
- - 免費遊戲1: 所有得分組合將額外乘上5倍。
- - 免費遊戲2: 每局皆有機會隨機出現2X或3X [WD]覆蓋在滾輪上。
- 滾輪上任意出現3、4或5個[SC]時，可再觸發免費遊戲10局。
