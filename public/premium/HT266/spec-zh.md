# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特色：圖標收集、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     | 描述       |
|-----------|------------|
| [H1]~[H4] | 高賠       |
| [L1]~[L5] | 低賠       |
| [WD]      | 百搭       |
| [SC]      | 分散       |
| [BO]      | 增強       |
| [B1]      | 獎勵遊戲 1 |
| [B2]      | 獎勵遊戲 2 |

### 百搭圖標
- [WD]出現在第2、3或4滾輪。
- [WD]可替代除了[SC]、[BO]、[B1]和[B2]以外的所有圖標。

### 分散圖標
- [SC]只出現在免費遊戲。
- 滾輪上任意位置出現3個或以上[SC]，即可獲得8場免費遊戲。

### 增強圖標
- 當[BO]出現時，滾輪上方的收集欄將+1x。

### 獎勵圖標
- 滾輪上任意位置出現6個或以上[B1]，即可獲得獎勵遊戲1。
- [B1]的隨機倍數為 0.1x、0.15x、0.2x、0.25x、0.5x、1x、1.5x、2x、2.5x、5x、10x。
- [B2]只出現在主遊戲。
- 滾輪上任意位置出現3個[B2]，即可獲得1場獎勵遊戲2。
---

## 遊戲特色

### 獎勵遊戲 1
- 當觸發獎勵遊戲時，觸發的[B1]將被鎖定在盤面上。
- 獎勵遊戲 1 中出現的 [B1] 將被鎖定。
- 當獎勵場次第一次結束時，會有3個選項可供選擇。選項是添加1、2或3獎勵場次。
- 當所有盤面已滿時，所有[B1]將隨機乘以X2或X3，並可能獎勵額外倍數25x、50X或1000x。
- 當剩餘旋轉為 0 或盤面已滿時，獎勵遊戲結束。
- 當獎勵遊戲結束時，收集欄將從滾輪的倍數到 [B1] 圖標並重置為 1x，並且所有 [B1] 倍數均被獎勵。

### 獎勵遊戲 2
- 獎勵遊戲 2 是一款輪盤遊戲。
- 輪盤獎勵包括得分積分、額外獎勵旋轉、免費遊戲或混合獎勵。

### 免費遊戲
- 滾輪上任意位置出現3個或以上[SC]，可再獲得8場免費遊戲。
- 其餘規則與主遊戲相同。
