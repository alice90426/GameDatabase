# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Collect Symbol, Free Game, Bonus Game.
---

## Symbols
|  Symbol   | Description  |
|-----------|--------------|
| [H1]~[H4] | High Pay     |
| [L1]~[L5] | Low Pay      |
| [WD]      | Wild         |
| [SC]      | Scatter      |
| [BO]      | Boost        |
| [B1]      | Bonus Game 1 |
| [B2]      | Bonus Game 2 |

### Wild Symbol
- [WD] appears on reels 2, 3 and 4 only.
- [WD] substitutes for all symbols except [SC], [BO], [B1] and [B2].

### Scatter Symbol
- Only appears in Free Game.
- When 3 or more [SC] symbols appear anywhere will retrigger the Free Game with 8 free spins.

### Boost Symbol
- When [BO] appears, the collection bar above that reel will be +1X.

### Bonus Symbol
- When 6 or more [B1] symbols appear anywhere will trigger the Bonus Game 1, the number of bonus spins is the same as the trigger count.
- [B1] have a random multiplier of 0.1x, 0.15x, 0.2x, 0.25x, 0.5x, 1x, 1.5x, 2x, 2.5x, 5x, 10x.
- [B2] only appears in Base Game.
- When 3 [B2] symbols appear anywhere will trigger the Bonus Game 2 with 1 bonus spin.
---

## Game Features

### Bonus Game 1
- When trigger Bonus Game, the triggered [B1] will be locked on reels.
- [B1] appearing in Bonus Game 1 will be locked.
- When bonus spins end first time, there will be 3 options to choose from. The options are add 1/2/3 bonus spins.
- When all reels are full, all [B1] will be random multiplied by X2 or X3, and may award additional multiplier 25X, 50X or 1000X.
- When remaining spins are 0 or reels are full, the Bonus Game ends.
- When Bonus Game ends, the collection bar will multi to [B1] symbols of that reel and reset to 1X, and all [B1] multipliers are awarded.

### Bonus Game 2
- Bonus Game 2 is a wheel game.
- The wheel awards include win points, extra bonus spin, Free Game or mixed rewards.

### Free Game
- [B2] will not appear in Free Game, and [SC] will appear.
- When 3 or more [SC] symbols appear anywhere will retrigger the Free Game with 8 free spins.
- The remaining rules are identical to the Base Game.
