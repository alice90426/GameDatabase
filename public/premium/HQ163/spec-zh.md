# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：圖標轉換、免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L6] | 低賠         |
| [WD]      | 百搭         |
| [W1]      | 轉換百搭     |
| [SC]      | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]出現在第2、3、4或5滾輪上。
- [W1]出現在第5滾輪上。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]即可獲得10、15、20場免費遊戲與2、10或100倍總押注得分。
---

## 遊戲特色

### 免費遊戲
- 當第5滾輪出現[W1]時，會將2、3、4滾輪上的高賠圖標變成[WD]。
- 滾輪上任意位置出現3、4或5個[SC]即可再獲得10、15、20場免費遊戲與2、10或100倍總押注得分。
