# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 功能：得分倍數、位移百搭、重轉、免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |

### 百搭圖標
- [WD] 替代除 [SC] 之外的所有圖標。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將觸發免費遊戲和 8/12/20 免費局數。
---

## 遊戲特色

### 得分倍數
- 當[WD]出現並有得分時，得分倍數將被激活，倍數為X2、X3或X5。

### 重轉
- 當主遊戲中出現超過2個[WD] 圖標時，將觸發重轉和1個重轉。
- 當重轉觸發時，[WD]將被保留並移動到隨機位置。 （[WD] 不會重疊）。
- 當重轉中出現新的[WD]時，將再觸發另外1個重轉，直到沒有新的[WD]。

### 免費遊戲
- 免費遊戲中出現的 [WD] 圖標將被保留並移動到隨機位置。
