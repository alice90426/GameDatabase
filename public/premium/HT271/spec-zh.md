# 遊戲資訊

## 簡介
- 盤面：5滾輪，5列，雙盤面。
- 線數：50線。
- 特色：收集圖標、百搭倍數、免費遊戲、獎勵遊戲。

### 雙盤面
- 主遊戲和免費遊戲中有兩個盤面。
- 第一個盤面是遊戲盤面，中心圖標始終是[WD]。
- 第二個盤面是賓果盤面，中心位置會觸發獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L5] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC] 和 [BN] 之外的所有圖標。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將觸發免費遊戲和 8/16/24 免費局數。

### 獎勵圖標
- [BN] 的隨機獎金為 0.1x、0.15x、0.2x、0.3x、0.5x、0.8x、1x、1.5x 2x、3x、5x、8x、10x、25x、50x 和 1000x。
- 出現在遊戲盤面中的 [BN] 將被添加到賓果盤面中。
---

## 遊戲特色

### 賓果線
- 當 [BN] 圖標在 2 日-盤面填寫任何賓果線時，將為該賓果線頒發獎品。
- 觸發賓果線後，賓果盤面會重置，5~10個[BN] 圖標會隨機放置在賓果盤面上。

### 獎勵遊戲
- 在賓果線中，包括中心位置將觸發獎勵遊戲和 6 獎勵局數。
- 獎勵遊戲中出現的 [BN] 將被鎖定。
- 當剩餘旋轉為 0 時，所有 [BN] 倍數加總成為中心倍數。

### 免費遊戲
- 中心 [WD] 將具有倍數 X2、X3、X5 或 X10。
- 當 3/4/5 [SC] 圖標出現在任何地方時，將在 8/16/24 免費局數再觸發免費遊戲。
- 其餘規則與主遊戲相同。
