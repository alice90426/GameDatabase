# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特色：巨型圖標、免費遊戲、獎勵遊戲。
---

## 圖標
| 圖標      | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L5] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |
| [B1]      | 倍數 |
| [B2]      | 增加 |
| [B3]      | 收集 |

### 百搭圖標
- [WD]可替代除了[SC]、[BN]、[B1]、[B2]和[B3]以外的所有圖標。

### 分散圖標
- [SC]出現在第2、3或4滾輪。
- 滾輪上任意位置出現3個[SC]，即可獲得6場免費遊戲。

### 獎勵圖標
- 滾輪上任意位置出現6個或以上[BN]，即可獲得6場獎勵遊戲。
- [BN] 具有 0.25x、0.5x、0.75x、1x、1.5x、2x、3x、5x、10x、20x、50x 或 100x 倍數。
- [B1] 具有 X2~X5 倍數。
- [B2] 具有 0.25x、0.5x、0.75x、1x、1.5x、2x、3x 或 5x 倍數。
---

## 遊戲特色

### 免費遊戲
- 每局出現3X3巨型圖標。
- 若出現巨型[SC]，則再觸發3局FG。
- 若出現巨型[BN]，則觸發6局BG2。巨型[BN]視為以9個[BN]觸發BG。

### 獎勵遊戲
-	BG使用4個盤面，初始解鎖第1個，其餘盤面依獲得金幣數量解鎖。
-	觸發BG的[BN]會鎖定在盤面上，若觸發局已滿足解鎖條件，則先解鎖再進行遊戲。
-	每次解鎖盤面可再觸發2局BG。
-	特殊圖標功能只對該盤面的[BN]有效，作用後會消失。
	- [BN1]:	隨機將一個盤面2的[BN]乘上2~5X。
	- [BN2]:	盤面3所有[BN]同時累加相同分數。
	- [BN3]:	獲得盤面4所有[BN]分數。
-	FG的巨型圖標相當於9個[BN]觸發BG，會直接解鎖盤面2並再觸發2局，再開始遊戲。
-	若所有盤面皆鎖滿[BN]，則提前結束BG。
