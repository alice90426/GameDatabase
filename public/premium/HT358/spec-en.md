# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 25 Lines.
- Features: Giant Symbol, Free Game, Bonus Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H4] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [SC]      | Scatter     |
| [BN]      | Bonus       |
| [B1]      | Multiplier  |
| [B2]      | Increase    |
| [B3]      | Collect     |

### Wild Symbol
- [WD] substitutes for all symbols except [SC], [BN] and [B1]~[B3].

### Scatter Symbol
- [SC] appears on reels 2, 3 and 4 only.
- When 3 [SC] symbols appear anywhere will trigger the Free Game with 6 free spins.

### Bonus Symbol
- When 6 or more [BN] symbols appear anywhere will trigger the Bonus Game with 6 bonus spins.
- The [BN] symbol has a multiplier of 0.25x, 0.5x, 0.75x, 1x, 1.5x, 2x, 3x, 5x, 10x, 20x, 50x or 100x.
- The [B1] symbol has a multiple of X2 to X5.
- The [B2] symbol has a multiplier of 0.25x, 0.5x, 0.75x, 1x, 1.5x, 2x, 3x or 5x.
---

## Game Features

### Free Game
- A 3x3 Giant Symbol appears on every spin.
- If a Giant [SC] appears, retrigger 3 Free Game spins.
- If a Giant [BN] appears, trigger 6 spins of Bonus Game 2. A Giant [BN] counts as 9 [BN] to trigger the Bonus Game.

### Bonus Game
- The Bonus Game uses 4 boards. Board 1 is unlocked at the start. The other boards are unlocked by the number of coins collected.
- Triggering [BN] symbols are locked on the boards. If the unlock condition is already met on the triggering spin, unlock the board first, then continue the Bonus Game.
- Each time a board is unlocked, 2 extra Bonus Game spins are awarded.
- Special symbol effects apply only to [BN] on that board and disappear after applying.
   - [BN1]: Randomly multiplies one [BN] on board 2 by X2 to X5.
   - [BN2]: All [BN] on board 3 gain the same added value.
   - [BN3]: Collects the values of all [BN] on board 4.
- The Giant Symbol in Free Game counts as 9 [BN] to trigger the Bonus Game. It will directly unlock board 2 and grant 2 extra Bonus Game spins, then the game starts.
- If all boards are fully locked with [BN], the Bonus Game ends early.
