# 遊戲資訊

## 簡介
- 盤面：3滾輪，3列。
- 線數：5線。
- 特色：獎金圖標、獎勵遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H4]   | 高賠         |
| [L1]~[L4]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的任意圖標來得到連線分數。

### 分散圖標
- 出現[SC]時，即可觸發3場獎勵遊戲與4倍的總押注得分。
- [SC]只出現在正中間滾輪。
---

## 遊戲特色

### 獎勵遊戲
- 滾輪上任意位置出現獎金圖標即可得分，且停留在滾輪上直到獎勵遊戲結束。
- 獎金圖標上可能出現的分數為: 200, 100, 60, 40, 30, 20, 10, 8, 5, 3, 2。
- 滾輪上集滿9個獎金圖標時，即可獲得32x/100x/1000x的總押注得分。
- 獎勵遊戲中，滾輪上任意位置出現1個或以上的獎金圖標，獎勵遊戲剩餘場次將重置為3場。
