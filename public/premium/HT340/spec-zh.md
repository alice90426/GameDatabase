# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：40線。
- 特色：免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC1]     | 分散 |
| [SC2]     | 倍數 |
| [BN1]     | 獎勵 |
| [BN2]     | 輪盤 |

### 百搭圖標
- [WD] 替代除 [SC] 和 [BN] 之外的所有圖標。

### 分散圖標
- 當任意位置出現 5 個或更多 [SC1] 或 [SC2] 圖標時將觸發免費遊戲。
- 免費局數等於觸發它的 [SC1] 或 [SC2] 的數量。
- [SC1] 和 [SC2] 帶有 0.1x、0.2x、0.5x、0.8x、1x、1.5x、2x、2.5x 或 3x 的倍數。

### 獎勵圖標
- 當 1 個或多個 [BN1] 出現在任何地方時，可能會觸發獎勵遊戲並進行 1 次獎勵旋轉。
---

## 遊戲特色

### 免費遊戲
- 盤面的底部將具有基於投注金額的初始值。
- [SC1] 或 [SC2] 上的值將被收集到相應的滾輪的底部中。
- 每個 [SC2] 都會將相應行的倍數增加 1，直至最大 X10。
- 免費遊戲結束後，獎勵遊戲將開始。

### 獎勵遊戲
- 初始獎勵自旋為 1，當新的圖標出現時，剩餘的自旋重置為 1。
- 每個[BN1]出現在盤面上，對應的滾輪的值將被添加到[BN1]上。
- [BN2] 帶有 5x、10x、50x 或 1000x 的倍數。
- 當旋轉為 0 或盤面已滿時，獎勵遊戲結束。
- 當獎勵遊戲結束時，[BN1] 或 [BN2] 上的所有值都將乘以相應行的倍數。
