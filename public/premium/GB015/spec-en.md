# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 50 Lines.
- Features: Giant Symbol, Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H5] | High Pay        |
| [L1]~[L5] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- 1 giant [SC] anywhere on the reels trigger 8 free spins.
---

## Game Features

### Free Game
- Giant symbols of [H1]~[H5], [WD], [SC] will appear on the reels.
- [L1]~[L5]will be removed from reels 1 to 5.
