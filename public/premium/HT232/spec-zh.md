# 遊戲資訊

## 簡介
- 盤面：5滾輪，4列。
- 線數：40線。
- 特色：倍數百搭、得分倍數、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L5] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |
| [W1]      | 乘數百搭     |
| [B1]      | 倍數滾輪     |
| [B2]      | 添加全部盤面 |
| [B3]      | 添加滾輪     |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。

### 倍數百搭
- [W1] 有 2x 到 200x 的隨機倍數。
- 在主遊戲中，帶有 [W1] 的獲勝組合將乘以倍數。
- 如果賠付線的得分中有多個 [W1] 圖標部分，則賠付線上所有 [W1] 圖標的倍數會相互相加。

### 獎勵圖標
- [B1]、[B2]或[B3]只出現在獎勵遊戲。
- [B1] 有 2x 到 25x 的隨機倍數。
- [B1] 會將其自己的倍數乘以高於滾輪的值。
- [B2]或[B3] 具有 1x 到 500x 的隨機倍數。
- [B2] 將在所有盤面之上添加自己的倍數。
- [B3] 將在盤面之上添加自己的倍數。
---

## 遊戲特色

### 免費遊戲
- 滾輪上任意位置出現3個[SC]，即可獲得10場免費遊戲。
- 滾輪上任意位置出現2、3或4個[SC]，即可獲得2、4或6場免費遊戲。
- [W1] 的所有倍數將被收集並累積在頂部，直到免費遊戲結束。
- 然後得分將乘以頂部的累積倍數。

### 獎勵遊戲
- 滾輪上任意位置出現4個[SC]，即可獲得3場獎勵遊戲。
- 如果出現新的圖標，旋轉次數將重置為 3。
- 獎勵遊戲結束後，將獎勵所有盤面之上的倍數之和。
