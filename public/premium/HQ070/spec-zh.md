# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特色：倍數百搭、圖標轉換、免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L6] | 低賠         |
| [WD]      | 百搭         |
| [W1]      | 倍數百搭     |
| [SC]      | 分散         |

### 百搭圖標
- [WD]或[W1]可替代除了[SC]以外的所有圖標。
- [WD]只出現在第2滾輪。
- [W1]只出現在第4滾輪。
- 得分組合上有出現[W1]，連線得分將額外乘上2倍。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得免費遊戲與2、10或300倍總押注得分。
- 點選[SC]，即可獲得隨機2、5或10場免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 免費遊戲中，滾輪上出現[WD]或[W1]時，可取代滾輪上任何大獎圖標。
- 滾輪上任意位置出現3、4或5個[SC]，可再獲得免費遊戲。
