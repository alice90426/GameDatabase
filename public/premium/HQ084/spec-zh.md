# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：30線。
- 特色：複製百搭、滑動百搭、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。
- [WD]出現在第1、2、4或5滾輪。
- 第1、2、4或5滾輪出現滿輪百搭，第5、4、2或1滾輪將會變成滿輪百搭。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得12場免費遊戲與1、5或25倍總押注得分。
---

## 遊戲特色

### 免費遊戲
- 免費遊戲中，滾輪上出現[WD]時，會將滾輪轉換為滿輪[WD]。
- 滾輪上任意位置出現3、4、5個[SC]即可再觸發12場免費遊戲。
