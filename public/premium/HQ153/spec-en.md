# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: Any 6.
- Features: Win Multiplier, Free Game.
---

## Symbols
|  Symbol     | Description     |
|-------------|-----------------|
| [H1]~[H4]   | High Pay        |
| [L1]~[L4]   | Low Pay         |
| [SC]        | Scatter         |

### Scatter Symbol
- 6 or more [SC] anywhere on the reels trigger 10 free spins.
---

## Game Features

### Free Game
- Any winning combination will multiply win multiplier.
- The multiplier is equal to the number of [SC] symbols that trigger the Free Game.
