# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：經典。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H5] | 高賠         |
| [L1]~[L5] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得2、10或100倍總押注得分。
