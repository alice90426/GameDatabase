# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：50線。
- 特點：三重能量、額外獎勵、額外盤面、額外旋轉、獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L5] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC] 之外的所有圖標。

### 分散圖標
- 僅出現在主遊戲中。

### 獎勵圖標
- 僅出現在獎勵遊戲中。
---

## 遊戲特色

### 獎勵遊戲
- 當 [SC] 圖標出現在任何地方時，可能會觸發具有 3 個獎勵局數和一個或多個功能的獎勵遊戲。
- 當正好觸發 1 個功能時，玩家可以選擇“保存”或“GO”。
  - SAVE：保留該功能，並在觸發另一個功能時觸發該功能。
  - GO：將觸發獎勵遊戲。
- 如果出現新的 [BN]，則獎勵局數將被重置，[BN] 將被鎖定。
- [BN] 具有 0.5x、1x、1.5x、2x、3x、5x、8x、10x、50x、300x 的隨機倍數。
- 當所有盤面全部滿時，玩家將獲得 2000x。
- 額外獎勵、額外盤面或額外旋轉可能同時觸發，共有7種類型獎勵遊戲。
  - 額外獎勵：0.5x、1x、1.5x、2x、3x、5x、8x 的隨機倍數可能會添加到鎖定的 [BN] 圖標中。
  - 額外盤面: [BN] 圖標將複製移動到額外盤面上的相應位置。
  - 額外旋轉獎勵局數更改為 4。
