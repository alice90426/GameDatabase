# 遊戲資訊

## 簡介
- 盤面：5滾輪，67776列。
- 線數：12348路。
- 特點：擴展滾輪、隨機圖標、重轉、免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L5] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD]出現在第2、3或4滾輪。
- [WD]可替代除了[SC]和[BN]以外的所有圖標。

### 分散圖標
- [SC]出現在第2、3或4滾輪。
- 滾輪上任意位置出現3個[SC]，即可獲得10場免費遊戲。

### 獎勵圖標
- [BN]只出現在第3滾輪。
- 當[BN]出現時，觸發1次重轉，並將盤面高度增加1行。
---

## 遊戲特色

### 隨機圖標
- 在主遊戲中，每次旋轉都有機會觸發該功能，該功能在盤面上出現 3 到 5 個 [H1]。

### 免費遊戲
- [H1] 的付款將加倍。
- 滾輪上任意位置出現3個[SC]，可再獲得10場免費遊戲。
- 滾輪高度保留到免費遊戲結束。
