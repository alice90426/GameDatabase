# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 25 Lines.
- Features: Prize Tower, Respin, Bonus Game, Free Game.
---

## Symbols
|  Symbol     | Description |
|-------------|-------------|
| [H1]~[H5]   | High Pay    |
| [L1]~[L3]   | Low Pay     |
| [WD]        | Wild        |
| [SC]        | Scatter     |
| [BN]        | Prize       |
| [BN1]       | Tower Count |
| [BN2]       | Bonus Game  |
| [BN3]       | Free Game   |
| [BN4]       | Upgrade     |
| [BN5]       | Prize       |
| [BN6]~[BN8] | Count 1~3   |

### Wild Symbol
- [WD] substitutes for all symbols except [SC], [BN1], [BN2], [BN3], [BN4], [BN5], [BN6], [BN7], [BN8] and [BN].

### Scatter Symbol
- When 1 to 4 [SC] symbols appear on the reels, the reel containing [SC] and the corresponding reel above will be locked, and respin once.

### Bonus Symbol
- When [BN2] appears above a reel with [SC] will trigger the Bonus Game with 3 bonus spins.
- [BN] or [BN5] carries a multiplier of 0.1x, 0.25x, 0.5x, 0.75x or 1x.
- [BN1], [BN2] or [BN3] is Tower Feature, Bonus Game or Free Game.
- [BN4], [BN5], [BN6], [BN7] or [BN8] appears in Bonus Game only.
---

## Game Features

### Prize Tower
- When tower count is 3 or more will award multiplier of 1x, 3x, 5x, 8x, 12x, 15x, 20x, 30x, 40x, 50x or 100x.

### Respin
- When Respin ends, all [BN], [BN1], [BN2] or [BN3] features above the [SC] will be awarded.
- Features can be combined, there are 3 Free Games and 4 Bonus Games.

### Bonus Game
- When 1 [BN2] appears above a reel with [SC] will trigger the feature with 3 bonus spins.
- When a new symbol appears, the remaining spins reset to 3.
- When all the reels are full, all rewards are doubled.
- When [BN4] appears, all [BN5] multiplier increase, and [BN6], [BN7] or [BN8] counts are doubled.
- If the feature is triggered with 2 [BN2], the reels will expand to 5 rows.
- If the feature is triggered with [BN2], the value on [BN2] becomes the base value of tower count.

### Free Game
- When 1 [BN3] appears above a reel with [SC] will trigger the feature with 3 free spins.
- An x3 multiplier area appears on the second row. When [BN] or [BN1] appears on the area, the value on symbols is multiplied by 3.
- If the feature is triggered with 2 [BN3], the multiplier area will expand to 3 rows.
- If the feature is triggered with [BN2], the value on [BN2] becomes the base value of tower count.
