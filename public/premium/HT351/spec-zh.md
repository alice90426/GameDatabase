# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特色：獎金塔、重轉、獎勵遊戲、免費遊戲。
---

## 圖標
|  圖標       | 描述     |
|-------------|----------|
| [H1]~[H5]   | 高賠     |
| [L1]~[L3]   | 低賠     |
| [WD]        | 百搭     |
| [SC]        | 分散     |
| [BN]        | 獎金     |
| [BN1]       | 塔數     |
| [BN2]       | 獎勵遊戲 |
| [BN3]       | 免費遊戲 |
| [BN4]       | 升級     |
| [BN5]       | 獎金     |
| [BN6]~[BN8] | 1~3 數量 |

### 百搭圖標
- [WD]可替代除了[SC]、[BN1]、[BN2]、[BN3]、[BN4]、[BN5]、[BN6]、[BN7]、[BN8]和[BN]以外的所有圖標。

### 分散圖標
- 當盤面上出現1到4個[SC] 圖標時，包含[SC]的滾輪和上面對應的滾輪將被鎖定，並且重轉一次。

### 獎勵圖標
- 當 [BN2] 出現在帶有 [SC] 的滾輪上方時，將觸發帶有 3 獎勵場次的獎勵遊戲。
- [BN]或[BN5] 帶有 0.1x、0.25x、0.5x、0.75x 或 1x 的倍數。
- [BN1]、[BN2]或[BN3] 是塔式功能、獎勵遊戲或免費遊戲。
- [BN4]、[BN5]、[BN6]、[BN7]或[BN8]只出現在獎勵遊戲。
---

## 遊戲特色

### 獎品塔
- 當塔數為3個以上時，將獎勵 1x、3x、5x、8x、12x、15x、20x、30x、40x、50x 或 100x 的倍數。

### 重轉
- 當重轉結束時，將授予 [SC] 以上的所有 [BN]、[BN1]、[BN2]或[BN3] 功能。
- 功能可以組合，有3個免費遊戲和4個獎勵遊戲。

### 獎勵遊戲
- 當 1 [BN2] 出現在帶有 [SC] 的滾輪上方時，將觸發帶有 3 獎勵場次的功能。
- 當新的圖標出現時，剩餘旋轉重置為 3。
- 當所有盤面滿時，所有獎勵加倍。
- 當 [BN4] 出現時，所有 [BN5] 倍數都會增加，並且 [BN6]、[BN7]或[BN8] 計數加倍。
- 如果使用 2 個 [BN2] 觸發該功能，則盤面將擴展到 5 行。
- 如果使用 [BN2] 觸發該功能，則 [BN2] 上的值將成為塔計數的基值。

### 免費遊戲
- 當 1 [BN3] 出現在帶有 [SC] 的滾輪上方時，將觸發帶有 3 免費場次的功能。
- X3 倍數區域出現在第二行。當該區域出現 [BN] 或 [BN1] 時，圖標上的值乘以 3。
- 如果該功能由 2 [BN3] 觸發，倍數區域將擴展到 3 行。
- 如果使用 [BN2] 觸發該功能，則 [BN2] 上的值將成為塔計數的基值。
