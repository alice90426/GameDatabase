# Game Information

## Introduction
- Size: 6 reels, 7 rows.
- Lines: 117649 Ways.
- Features: Wild Reel, Select Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H5] | High Pay    |
| [L1]~[L4] | Low Pay     |
| [WD]      | Wild        |
| [SC]      | Scatter     |

### Wild Symbol
- [WD] appears on reels 2, 3, 4 and 5 only.
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- [SC] appears on reels 2, 3, 4 and 5 only.
- When 3 [SC] symbols appear anywhere will trigger the Select Free Game.
---

## Game Features

### Wild Reel
- Base Game will randomly trigger the feature. 1 to 3 full-reel [WD] symbols randomly placing on the reels.

### Select Free Game
- Free Game has 3 modes to choose from, the free spins and full-reel [WD] symbols are different.
  - Free Game 1: 12 free spins with 1 full-reel [WD] symbol.
  - Free Game 2: 8 free spins with 2 full-reel [WD] symbol.
  - Free Game 3: 3 free spins with 3 full-reel [WD] symbol.
