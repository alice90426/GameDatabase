# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243線。
- 特色：圖標升級、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H4]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 升級百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只會出現在2、3、4滾輪。
- [W1]只出現在免費遊戲第5輪。

### 分散圖標
- 第2、3、4滾輪上出現3[SC]即可觸發12場免費遊戲與2倍的總押注得分。
---

## 遊戲特色

### 免費遊戲
- 每2個[W1]可將一種高賠圖標升級成[WD]。
- 只有2~5滾輪的圖標可以升級。
- 第2、3、4滾輪上出現3[SC]即可再觸發12場免費遊戲與2倍的總押注得分。
---
