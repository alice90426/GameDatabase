# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243線。
- 特色：圖標升級、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H4]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 升級百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]或[W1]可替代除了[SC]以外的所有圖標。
- [WD]出現在第2、3或4滾輪。
- [W1]只出現在免費遊戲中的第5滾輪。

### 分散圖標
- 第2、3或4滾輪上任意位置出現3個[SC]，即可獲得12場免費遊戲與2倍總押注得分。
---

## 遊戲特色

### 免費遊戲
- 每2個[W1]可將一種高賠圖標升級成[WD]。
- 只有第2、3、4或5滾輪的圖標可以升級。
- 第2、3或4滾輪上任意位置出現3個[SC]，可再獲得6場免費遊戲與2倍總押注得分。
---
