# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Win Multiplier, Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H4] | High Pay        |
| [L1]~[L4] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- All Reels with [SC] trigger 13 free spins.
---

## Game Features

### Free Game
- Any winning combination will multiply from 3X up to 72X!
- Player can retrigger 13 free spins again when All Reels with [SC].
