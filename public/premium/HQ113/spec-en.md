# Game Information

## Introduction
- Size: 5 reels, 35553 rows.
- Lines: 7776 Ways.
- Features: Expand Reel, Select Free Game.
---

## Symbols
|  Symbol     | Description     |
|-------------|-----------------|
| [H1]~[H5]   | High Pay        |
| [L1]~[L6]   | Low Pay         |
| [WD]        | Wild            |
| [W1]        | Super Wild      |
| [SC]        | Scatter         |

### Wild Symbol
- [WD] or [W1] substitutes for all symbols except [SC].
- [WD] appears on reels 2, 3 or 4.
- [W1] appears on reel 3 only.

### Scatter Symbol
- 3, 4 or 5 [SC] from leftmost to right on adjacent reels trigger Select Free Game, and win 5x, 10x or 50x total play amount.
---

## Game Features

### Expand Reel
- The game include 3 levels. Each stage has different reel size: 33333->34443->35553 rows.
- On level 1 and 2, [W1] appears anywhere on the reels make reel 2, 3 and 4 grow 1 size on the spin.

### Select Free Game
- Five featured Free Game with different reel size and spin times can be chosen when a Free Game be triggered.
- 2 or more [SC] from leftmost to right on adjacent reels retrigger 3 free spins.
