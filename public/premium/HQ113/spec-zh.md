# 遊戲資訊

## 簡介
- 盤面：5滾輪，35553列。
- 線數：7776路。
- 特色：延展滾輪、多選項免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L6]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 超級百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只出現在第2、3和4滾輪。
- [W1]只出現在第3滾輪。

### 分散圖標
- 滾輪上由最左至右連續出現3、4或5個[SC]時，可獲得免費遊戲與5、10或50倍的總押注得分。
---

## 遊戲特色

### 延展滾輪
- 遊戲共分為三關，三關滾輪大小各自不同: 33333->34443->35553 列。
- 第1、2關當中，[W1]出現在盤面上會觸發第2、3、4滾輪延展一格。

### 多選項免費遊戲
- 特色選項為五種不同滾輪高度與免費遊戲場次。
- 免費遊戲中滾輪上由最左至右連續出現2個以上[SC]時，可再獲得3場免費遊戲。
