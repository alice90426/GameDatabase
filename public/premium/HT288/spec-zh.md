# 遊戲資訊

## 簡介
- 盤面：6滾輪，5列。
- 線數：50線。
- 特色：倍數百搭、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     |  描述    |
|-----------|----------|
| [H1]~[H5] | 高賠     |
| [H6]~[H7] | 中賠     |
| [L1]~[L4] | 低賠     |
| [WD]      | 百搭     |
| [SC]      | 分散     |
| [BN]      | 獎勵     |
| [B1]      | 十字     |
| [B2]      | 輪盤獎   |
| [B3]      | 完整收集 |

### 百搭圖標
- [WD]出現在第2、3、4或5滾輪。
- [WD]可替代除了[SC]和[BN]以外的所有圖標。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得免費遊戲或獎勵遊戲。

### 獎勵圖標
- 滾輪上任意位置出現4個或以上[BN]，即可獲得5場獎勵遊戲。
- [BN] 將具有隨機點：0.1x、0.2x、0.3x、0.5x、0.8x、1x、2x、3x。
- [B1]或[B2]只出現在獎勵遊戲。
- 當 [B1] 出現時，所有交叉路徑將增加 0.1x、0.2x、0.3x 或 0.5x，並向該滾輪添加 1 個旋轉。
- 當 [B2] 出現時，即可獲得隨機獎勵：3x、3.5x、4x、5x、10x、50x、1000x，並在滾輪上添加 2 次旋轉。
---

## 遊戲特色

### 免費遊戲
- 當免費遊戲被觸發時，將得到一個輪盤來確定免費場次和[WD]倍數。
  - 免費場次：6、9、12 或 15。
  - [WD] 倍數：X2、X3、X5 或 X8。
- 免費遊戲中的所有 [WD] 將攜帶倍數。
- 當任意位置出現 3 個或以上 [SC] 圖標時，將再觸發免費遊戲，且免費場次與觸發免費場次相同。

### 獎勵遊戲
- 雙盤面將出現在獎勵遊戲中，每個滾輪有 5 次旋轉。
- 如果出現新的 [BN]，[BN] 將被鎖定。
- 當一個滾輪滿時，即可獲得隨機倍數和額外旋轉，並且所有 [BN] 圖標將合併到 [B3] 並返回到滾輪。
  - 隨機倍數：X2、X3 或 X10，僅完整滾輪。
  - 額外旋轉：+2、+3 或 +5，兩個盤面都會添加額外旋轉。
