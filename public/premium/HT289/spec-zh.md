# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特色：倍數百搭、免費遊戲、獎勵輪盤。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD]出現在第2、3或4滾輪。
- [WD]可替代除了[SC]和[BN]以外的所有圖標。
- [WD] 可能與倍數 X2 或 X3 一起出現。

### 分散圖標
- [SC]出現在第3、4或5滾輪。
- 滾輪上任意位置出現3個[SC]，即可獲得9場免費遊戲。

### 獎勵圖標
- [BN]出現在第2、3或4滾輪。
- 當[BN]出現時，滾輪對應的獎勵輪盤將會旋轉。
---

## 遊戲特色

### 免費遊戲
- 免費遊戲中還有更多 [BN] 圖標。
- 其餘規則與主遊戲相同。

### 獎勵輪盤
- 獎勵滾輪可能出現輪盤圖標或 0、1、2、3、5、10、00。 得分點等於數字 * 0.1 總投注。
- 輪盤圖標可能會獎勵 5 倍、6 倍、8 倍、10 倍、20 倍、50 倍或 200 倍的倍數。
