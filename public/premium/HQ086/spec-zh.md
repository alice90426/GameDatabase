# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特色：額外滾輪、得分倍數、延展百搭、隨機百搭、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |
| [HW]        | 超級百搭     |

### 百搭圖標
- [WD]可替代除了[SC]以外的任意圖標來得到連線分數。
- 當[WD]位於滾輪中間時將展開成滿輪[WD]。

### 分散圖標
- 滾輪上任意位置出現3個[SC]即可觸發10場免費遊戲與1倍的總押注得分。
- [SC]只出現在第1、3、5滾輪。
---

## 遊戲特色

### 額外滾輪
- 遊戲中共有6個滾輪，第6個滾輪會出現額外倍數。
- 得分倍數:
- - 主遊戲: 1X~10X
- - 免費遊戲: 2X~10X

### 免費遊戲
- [HW]只出現在免費遊戲第6滾輪，並可觸發隨機百搭特色，觸發時每個滾輪將會隨機出現1個[WD]。
- 滾輪上任意位置出現3個[SC]即可再觸發10場免費遊戲。
