# 遊戲資訊

## 簡介
- 盤面：6滾輪，7列。
- 線數：117649路。
- 特色：消除滾輪、最大路數、獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H4] | 高賠 |
| [L1]~[L6] | 低賠 |
| [WD]      | 百搭 |
| [BN]      | 獎勵 |

### 百搭圖標
- [WD]出現在第2、3、4或5滾輪。
- [WD]可替代除了[BN]以外的所有圖標。

### 獎勵圖標
- [BN] 具有 1x、2x、3x、4x、5x、10x、20x、50x、100x、200x、500x、1000x 的隨機倍數。
---

## 遊戲特色

### 獎勵遊戲
- 滾輪上任意位置出現5個或以上[BN]，即可獲得3場獎勵遊戲。
- 當觸發獎勵遊戲時，觸發的[BN]和當前滾輪高度將被鎖定。
- 如果出現新的[BN]，旋轉次數將重置為3，並且[BN]將被鎖定。
- 當滾輪已滿時，該滾輪中的所有 [BN] 都會將 X2 乘以 X10。
- 當剩餘旋轉為 0 或盤面已滿時，獎勵遊戲結束，所有 [BN] 倍數均被獎勵。

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 最大路數
- 主遊戲將隨機觸發該功能。所有滾輪高度必須為 7。
