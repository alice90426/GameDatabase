# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 25 Lines.
- Features: Symbol Transform, Multiplier Wild, Symbol Collect, Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H5] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [SC]      | Scatter     |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- When 3 [SC] symbols appear anywhere will trigger the Free Game with 8 free spins. Each additional [SC] symbol awards 1 extra free spin.
---

## Game Features

### Symbol Transform
- When a symbol on the Transformation-Reel matches the Main-Reel, the matching symbol on the Main-Reel will be converted in to [WD].
- When 2 to 5 identical symbols appear on the Transformation-Reel, the corresponding symbol on the Main-Reel will be converted into a X2 to X5 [WD].

### Free Game
- Each [SC] awards 1 extra free spin.
- When Free Game is triggered, the transformed [WD] symbols will be recorded and remain active until Free Game ends.
- If the Transformation-Reel repeatedly displays the same symbol, the transformed [WD] will gain additional multiplier, up to maximum X5.
