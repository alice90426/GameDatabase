# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：消除特色、得分倍數、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L5] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |
| [BN]      | 獎勵         |

### 百搭圖標
- [WD]可替代除了[SC]與[BN]以外的任意圖標來得到連線分數。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，可獲得2、5或50倍總押注得分。
- [SC]只在獎勵遊戲中出現。

### 獎勵圖標
- 第2、4滾輪上最左至右連續出現6個[BN]，玩家即可獲得特色遊戲以及2倍的總押注得分。
- [BN]指出現在主遊戲中。
---

## 遊戲特色

### 消除特色
- 任何得分都會觸發消除特色，並且重複贏分直到沒有任何贏分為止。

### 免費遊戲
- 任何得分都會觸發消除特色，並將得分額外乘上一個倍數。
- 觸發消除特色次數越多，額外乘上的倍數越高，倍數從1X、2X、3X、5X，最高到10X。
- 免費遊戲固定為20場。

### 獎勵遊戲1
- 獎勵從1X、2X、6X、12X、18X、25X、40X、70X、150X、250X、350X、600X，最高到1000X。

### 獎勵遊戲2
- 獎勵從1X、10X、30X、100X、300X，最高到800X。
