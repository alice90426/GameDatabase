# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H5] | 高賠         |
| [L1]~[L6] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的所有圖標。
- [WD]出現在第2、3或4滾輪。

### 分散圖標
- 滾輪上由最左至右連續出現3、4或5個[SC]，即可獲得10場免費遊戲與5、10或50倍總押注得分。
---

## 遊戲特色

### 免費遊戲
- 滾輪上由最左至右連續出現3、4或5個[SC]，可再獲得10場免費遊戲。
- 免費遊戲中不會出現[L1]~[L6]。
