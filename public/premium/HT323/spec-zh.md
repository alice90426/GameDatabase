# 遊戲資訊

## 簡介
- 盤面：5滾輪，7列。
- 線數：117649路。
- 特色：消除掉落、隨機百搭、倍數百搭、免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L5] | 低賠 |
| [WD]      | 百搭 |
| [W1]      | 隨機 |
| [W2]      | 倍數 |
| [SC]      | 分散 |

### 百搭圖標
- [WD]/[W1]/[W2] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [W1]、[W2] 和 [SC] 之外的所有圖標。
- [W1] 替代除 [W2] 和 [SC] 之外的所有圖標。
- [W2] 替代除 [W1] 和 [SC] 之外的所有圖標。

### 分散圖標
- 當 3 [SC] 圖標出現在任意位置時，將觸發免費遊戲和 8 免費局數。每增加一個 [SC]，就添加 2 個免費局數。
---

## 遊戲特色

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 隨機百搭
- 當[W1]出現時，盤面 2到5上的圖標的隨機數將轉換為[WD] 圖標，不包括[SC]。

### 倍數百搭
- 當[W2]出現時，它會將自身及其上下1到3個隨機圖標變換為倍數[WD]。
- 倍數為X2、X3或X5。

### 免費遊戲
- 當 4 個或更多 [SC] 圖標出現在任何地方時，將用 5 個免費局數再觸發免費遊戲。
- 每次旋轉都會觸發隨機百搭功能。
