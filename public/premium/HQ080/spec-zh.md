# 遊戲資訊

## 簡介
- 盤面：5滾輪，34543列。
- 線數：720線。
- 特色：黏性百搭、重轉特色、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H3]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |
| [FG1]/[FG2] | 免費遊戲     |

### 百搭圖標
- [WD]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只會出現在2、3、4滾輪。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得1、5或30倍的總押注得分。

### 免費遊戲圖標
- 滾輪上由最左至右連續出現5個[FG1]或[FG2]，即可觸發免費遊戲1/2。
---

## 遊戲特色

### 免費遊戲
- [WD]和[FG1]/[FG2]會鎖定在特定位置。
- 免費遊戲中，任何連線得分皆會觸發重轉。
