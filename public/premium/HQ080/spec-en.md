# Game Information

## Introduction
- Size: 5 reels, 34543 rows.
- Lines: 720 Ways.
- Features: Sticky Wild, Respin, Free Game.
---

## Symbols
|  Symbol     | Description     |
|-------------|-----------------|
| [H1]~[H3]   | High Pay        |
| [L1]~[L5]   | Low Pay         |
| [WD]        | Wild            |
| [SC]        | Scatter         |
| [FG1], [FG2]| Free Game       |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].
- [WD] appears on reels 2, 3 or 4.

### Scatter Symbol
- 3, 4 or 5 [SC] anywher on the reels win 1x, 5x or 30x total play amount.

### Free Game Symbol
- 5 [FG1] or [FG2] from leftmost to right on adjacent reels trigger Free Game 1 or Free Game 2.
---

## Game Features

### Free Game
- [WD] and [FG1] or [FG2] lock on specific position.
- Any winning combination trigger respin.
