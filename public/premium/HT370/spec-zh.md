# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：50線。
- 特色：倍數百搭、免費遊戲、獎勵遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L6] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |
| [BN]      | 獎勵 |
| [B1]      | 倍數 |

### 百搭圖標
- [WD] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC]、[BN] 和 [B1] 之外的所有圖標。

### 分散圖標
- 當任意位置出現 3 個或更多 [SC] 圖標時，將觸發 10 個免費局數的免費遊戲。

### 獎勵圖標
- 當任意位置出現 3 個或更多 [BN] 圖標時，將觸發 5 個獎勵局數的獎勵遊戲。
- [BN] 圖標的倍數為 0.5x、1x、1.5x 或 2x。
- 當滾輪 5 出現 [WD] 的完整滾輪時，將授予 [B1] 上的所有獎勵。
- [B1] 圖標的倍數為 0.1x、0.5x、1x、1.5x、2x、3x、4x、5x、8x 或 10x。
---

## 遊戲特色

### 免費遊戲
- [L1]~[L6] 圖標在免費遊戲中被刪除。
- 當盤面 1 和 5 上的 [SC] 觸發免費遊戲 2 和 4 免費局數時。

### 免費遊戲 2
- 滾輪 3 是倍數 [WD] 圖標。
  - 在第一次和第二次旋轉時，X2 [WD] 圖標將出現在滾輪 3 上。
  - 在第 3 次旋轉時，X3 [WD] 圖標將出現在滾輪 3 上。
  - 在第 4 次旋轉時，X5 [WD] 圖標將出現在滾輪 3 上。

### 獎勵遊戲
- 每個滾輪上方都會出現一個獎池，並隨著每次旋轉而增加。
- 每個獎池都有不同的最大得分。一旦達到最大，獎池將不再增加。
- 當全部滾輪填滿[BN]時，相應獎池將被鎖定並頒發。
- 當 [SC] 圖標出現在任何地方時，都會用 1、2、3 或 5 獎勵局數再觸發獎勵遊戲。
- 獎金級別為 2x、3x、4x、5x、8x、10x、15x、20x、30x、50x、80x、100x 或 200x。
