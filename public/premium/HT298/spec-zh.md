# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特色：圖標收集、免費遊戲、獎勵輪盤。
---

## 圖標
|  圖標     | 描述   |
|-----------|--------|
| [H1]~[H5] | 高賠   |
| [L1]~[L6] | 低賠   |
| [WD]      | 百搭   |
| [SC]      | 分散   |
| [SC1]     | 金分散 |
| [BN]      | 獎勵   |

### 百搭圖標
- [WD]可替代除了[SC]、[SC1]和[BN]以外的所有圖標。
- [WD]出現在第2、3或4滾輪。

### 分散圖標
- 滾輪上任意位置出現3個[SC]或[SC1]，即可獲得獎勵輪盤。
- [SC1]只出現在第5滾輪。

### 獎勵圖標
- 滾輪上任意位置出現6個或以上[BN]，即可獲得6場免費遊戲。
---

## 遊戲特色

### 免費遊戲
- 當[BN]出現時，該位置將是標記作為LV1。
- 當[BN]出現在LV1位置時，該位置將升級1級，最高等級為LV3。
- 當免費遊戲結束時，那些標記位置將獎勵積分。
  - LV1：0.5x、1x、1.2x 或 1.8x
  - LV2：2x、3x、4x、5x、6x 或 10x。
  - LV3：5x、10x、15x、20x、40x 或 50x。

### 獎勵輪盤
- 獎勵輪盤可能會獎勵倍數或免費遊戲。
- 倍數是 5x、10x、20x、40x 或 100x 之一。
- 免費遊戲可能具有不同的功能。
  - 巨型圖標：在免費遊戲開始之前，2x2 到 3x5 巨型 [BN] 將出現在盤面和標記上。
  - 最高等級：標記位置將直接位於LV3。
  - 擴展盤面： 盤面擴展至 6x5。
