# 遊戲資訊

## 簡介
- 盤面：5滾輪，5列，雙盤面。
- 線數：6250路。
- 特點：隨機百搭、百搭滾輪、位移百搭、無限旋轉、免費遊戲、獎勵遊戲。

### 雙盤面
- 主遊戲和免費遊戲中有兩個盤面。
---

## 圖標
|  圖標     | 描述      |
|-----------|-----------|
| [H1]~[H5] | 高賠      |
| [L1]~[L6] | 低賠      |
| [WD]      | 百搭      |
| [SC]      | 分散      |
| [BN]      | 獎勵      |
| [B1]      | 收集      |
| [B2]      | 收集和提升 |

### 百搭圖標
- [WD]可替代除了[SC]和[BN]以外的所有圖標。

### 分散圖標
- [SC]只出現在主遊戲。
- 滾輪上任意位置出現3個或以上[SC]，即可獲得無限場免費遊戲。

### 獎勵圖標
- [BN]只出現在主遊戲。
- 滾輪上任意位置出現3個或以上[BN]，即可獲得3場獎勵遊戲。
- [B1] 將獲得收集 1 個獎項。
- [B2] 將提升所有獎項和收集 1 個獎項。
---

## 遊戲特色

### 隨機百搭
- 主遊戲將隨機觸發該功能。 [WD] 圖標隨機放置在盤面上。

### 百搭滾輪
- 當完整的[H1]到[H4]出現在滾輪中時，滾輪將轉換為 [WD] 滾輪，並將複製轉換為另一個盤面。
- 當所有盤面都是[WD]盤面時，將額外獲得200x。

### 免費遊戲
- 在免費遊戲的開頭，[WD]滾輪將出現在滾輪5的第2-盤面處。
- 如果沒有獲勝通過[WD]滾輪，它將向左移動一個滾輪。如果[WD]滾輪出現在第2-盤面處的滾輪 1中，它將移動到第1-盤面處的滾輪 5。
- 當[WD]滾輪退出盤面時，免費遊戲結束。

### 獎勵遊戲
- 獎勵遊戲中出現的[B1]和[B2]將被鎖定。
- [B1]和[B2]各將獲得1個獎項。
- 獎項為 1x、2x、3x、4x、5x、10x、100x、200x。
- 提升量依序為 1x、1x、1、1x、4x、10x、18x、30x。
