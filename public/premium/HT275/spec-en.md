# Game Information

## Introduction
- Size: 5 reels, 5 rows, Dual-Reels.
- Lines: 6250 Ways.
- Features: Random Wild, Wild Reel, Walking Wild, Infinite Spins, Free Game, Bonus Game.

### Dual-Reels
- There are two reels in Base Game and Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H5] | High Pay        |
| [L1]~[L6] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |
| [BN]      | Bonus           |
| [B1]      | Collect         |
| [B2]      | Collect & Boost |

### Wild Symbol
- [WD] substitutes for all symbols except [SC] and [BN].

### Scatter Symbol
- Only appears in Base Game.
- When 3 or more [SC] symbols appear anywhere in one reels will trigger the Free Game with infinite free spins.

### Bonus Symbol
- Only appears in Base Game.
- When 3 or more [BN] symbols appear anywhere in one reels will trigger the Bonus Game with 3 bonus spins.
- [B1] will collect 1 award.
- [B2] will boost all awards and collect 1 award.
---

## Game Features

### Random Wild
- Base Game will randomly trigger the feature. [WD] symbols randomly placing on the reels.

### Wild Reel
- When full [H1] to [H4] symbols appear in a reel, that reel will transform to [WD] reel, and duplicated to another reels.
- When all reels are [WD] reels, player will award 200X.

### Free Game
- At the start of the Free Game, a [WD] reel will appear in reel 5 at 2nd-Reels.
- If no winning passes through the [WD] reel, it will move one reel left. If [WD] reel appears in reel 1 at 2nd-Reels, it will move to reel 5 at 1st-Reels.
- The Free Game ends when the [WD] reel exits the reels.

### Bonus Game
- [B1] and [B2] appearing in Bonus Game will be locked.
- Each [B1] and [B2] will gain 1 award.
- Awards are 1x, 2x, 3x, 4x, 5x, 10x, 100x, 200x.
- Boosts are 1x, 1x, 1, 1x, 4x, 10x, 18x, 30x, is corresponded to each award.
