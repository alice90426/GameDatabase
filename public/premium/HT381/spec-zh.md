# 遊戲資訊

## 簡介
- 盤面：6滾輪，6列。
- 線數：46656路。
- 特色：消除滾輪、黏性百搭、免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H6] | 高賠 |
| [L1]~[L6] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |

### 百搭圖標
- [WD]出現在第2、3、4或5滾輪。
- [WD]可替代除了[SC]以外的所有圖標。

### 分散圖標
- 滾輪上任意位置出現3個[SC]，即可獲得8場免費遊戲。每額外1個[SC]增加2場免費遊戲。
---

## 遊戲特色

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 黏性百搭
- 該功能將在隨機中觸發主遊戲並始終在免費遊戲中觸發。
- 當該功能被觸發時，所有 [WD] 圖標將有 1、2 或 3 生命。
- 當[WD]連接時，生命減少1，當生命為0時[WD]會消失。

### 免費遊戲
- 滾輪上任意位置出現3個[SC]，可再獲得8場免費遊戲。每額外1個[SC]增加2場免費遊戲。
- 黏性百搭功能將觸發每次免費旋轉。
