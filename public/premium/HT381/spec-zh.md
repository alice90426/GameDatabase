# 遊戲資訊

## 簡介
- 盤面：6滾輪，6列。
- 線數：46656路。
- 特色：消除掉落、黏性百搭、免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H6] | 高賠 |
| [L1]~[L6] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |

### 百搭圖標
- [WD] 僅出現在盤面 2、3、4 和 5 上。
- [WD] 替代除 [SC] 之外的所有圖標。
---

## 遊戲特色

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 黏性百搭
- 該功能將在隨機中觸發主遊戲並始終在免費遊戲中觸發。
- 當該功能被觸發時，所有 [WD] 圖標將有 1、2 或 3 生命。
- 當[WD]連接時，生命減少1，當生命為0時[WD]會消失。

### 免費遊戲
- 3 分散圖標出現在任何地方都會用 8 免費局數觸發/再觸發免費遊戲。每增加一個 [SC]，就添加 2 個免費局數。
- 黏性百搭功能將觸發每次免費旋轉。
