# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：消除特色、得分倍數、免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H5] | 高賠         |
| [L1]~[L4] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的任意圖標來達到連線分數！
- [WD]堆疊在3、4、5滾輪。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，玩家即可獲得15、20或25場免費遊戲，且獲得5、50或250倍的總押注得分。
---

## 遊戲特色

### 消除特色
- 任何得分都會觸發消除特色，並且重複"消除特色"直到沒有任何得分為止。

### 免費遊戲
- 任何得分都會觸發消除特色，並將得分額外乘上一個倍數。
- 觸發消除特色次數越多，額外乘上的倍數越高。倍數從2X、3X、5X、8X，最高到10X。
