# 遊戲資訊

## 簡介
- 盤面：5滾輪，34443列。
- 線數：576路。
- 特色：倍數百搭、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L6]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 倍數百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]/[W1]只出現在第2、3和4滾輪。

### 分散圖標
- 滾輪上連續出現3個[SC]時，即可獲得免費遊戲和2倍的總押注得分。
- 點選[SC]可隨機獲得2、5或10場免費遊戲。
- 3個[SC]加總即為免費遊戲場次。
- [SC]只出現在第2、3和4滾輪。
---

## 遊戲特色

### 免費遊戲
- [WD]會隨機出現2X、3X或5X。
- 滾輪上連續出現3個[SC]時，即可再獲得免費遊戲和2倍的總押注得分。
