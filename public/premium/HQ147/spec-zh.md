# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：25線。
- 特色：倍數百搭、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [SC]        | 分散         |

### 百搭圖標
- [WD]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只出現在第2、3、4滾輪。
- 得分線上有出現[WD]，該線得分將額外乘上隨機倍數。當出現多個[WD]時，倍數會相加。
- 主遊戲倍數: 1X~3X。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]時，即可獲得4、5或6場免費遊戲與5、20或50倍的總押注得分。
- [SC]只出現在第3、4、5滾輪。
---

## 遊戲特色

### 免費遊戲
- 百搭倍數從1倍開始，每一場免費遊戲增加1倍。
- 免費遊戲中出現任意得分時，剩餘場次將會重置回觸發時的場次。
