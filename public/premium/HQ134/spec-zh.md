# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：倍數百搭、得分倍數、免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L6]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 倍數百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]或[W1]可替代除了[SC]以外的所有圖標。
- [WD]只出現在第3、4或5滾輪。
- [W1]可能出現2X、3X。
- 高賠圖標的5連線得分上有出現[WD]或[W1]時，即可獲得5場免費遊戲。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]時，可獲得2、5或10倍的總押注得分。
---

## 遊戲特色

### 免費遊戲
- 高賠圖標的5連線得分上有出現[WD]或[W1]時，即可再獲得5場免費遊戲。
- 免費遊戲中，滾輪上任意位置出現[SC]，即可獲得獎勵。獎勵與觸發此免費遊戲的連線分數相同。
---
