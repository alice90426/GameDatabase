# 遊戲資訊

## 簡介
- 盤面：6滾輪，5列。
- 線數：任意8。
- 特色：消除掉落、得分倍數、重轉、免費遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H5] | 高賠     |
| [L1]~[L4] | 低賠     |
| [BN]      | 倍數     |
| [B1]~[B7] | 獎勵圖標 |

### 獎勵圖標
- 3 [BN] 圖標任何地方出現時都會觸發重轉。
- 當任意位置出現 4 個或更多 [BN] 圖標時，將觸發獎勵遊戲和 3 個獎勵局數。
- 隨機攜帶 1x、2x、3x、4x、5x、6x、8x、10x、12x、15x、20x、25x、50x、100x、250x 或 500x 的倍數。
---

## 遊戲特色

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 重轉
- 3 [BN] 圖標的倍數將相加並存儲。
- 重新旋轉將繼續，直到出現獲勝組合。
- 當前回合的所有獎金將乘以存儲的倍數。

### 獎勵遊戲
- 如果盤面上出現新的圖標，則旋轉計數重置為 3。
- 盤面 1 和 6 最初被鎖定。當 [B4] 出現時，將額外解鎖 1 個滾輪，最多 2 個盤面。
- 僅會出現 [BN]、[B1]~[B7] 或空白圖標。他們的倍數將被累積。
  - [B1]：隨機帶有 1x、2x 或 3x 的倍數。該倍數將被添加到盤面上的所有圖標倍數一次。
  - [B2]：隨機帶有 X2、X3、4x 或 X5 的倍數。該倍數將在盤面上乘以 2~5 圖標一次。
  - [B3]：收集盤面上所有圖標的倍數一次。
  - [B5]：隨機帶有 1x、2x 或 3x 的倍數。該倍數將添加到每次旋轉盤面的所有圖標倍數中。
  - [B6]：隨機帶有 X2、X3、4x 或 X5 的倍數。每次旋轉時，該倍數將在盤面上乘以 2~5 圖標。
  - [B7]：收集每次旋轉盤面上所有圖標的倍數。
- 當剩餘旋轉達到 0 或盤面被填滿時，獎勵遊戲結束。
