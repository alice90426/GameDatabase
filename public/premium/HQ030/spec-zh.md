# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：得分倍數、免費遊戲。
---

## 圖標
|  圖標     |  描述        |
|-----------|--------------|
| [H1]~[H4] | 高賠         |
| [L1]~[L6] | 低賠         |
| [WD]      | 百搭         |
| [SC]      | 分散         |
| [BN]      | 獎勵         |

### 百搭圖標
- [WD]可替代除了[SC]和[BN]以外的任意圖標來達到連線分數！

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，玩家即可獲得3、10或50倍的總押注得分。

### 獎勵圖標
- 第1、2、3滾輪上由最左至右連續出現3個[BN]，玩家即可獲得免費遊戲以及2倍的總押注得分。
---

## 遊戲特色

### 免費遊戲
- 免費遊戲可獲得2X到6X的倍數和8到18場的免費遊戲。
- 免費遊戲中，第1、2、3滾輪上由最左至右連續出現3個[BN]，可再獲得免費遊戲。
