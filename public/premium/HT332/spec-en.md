# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Random Prize, Expanding Reels, Bonus Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]      | High Pay    |
| [H2]~[H7] | Low Pay     |
| [WD]      | Wild        |
| [BN]      | Digit       |
| [B1]      | Respin      |
| [B2]      | Expand      |

### Bonus Symbol
- When 2 or more [BN] symbols adjacent to each other on the same row, the [BN] will appear up to 2 digits.
- The digits multiplied by 0.001 total bet will be the final prize.
- When [B1] or [B2] appearing anywhere may trigger the Bonus Game with corresponding feature.
---

## Game Features

### Bonus Game
- The [BN] symbols will be locked on reels and transform to digits when Bonus Game ends.
- [B1]~[B3] may trigger at the same time, there are 3 type Bonus Games.
  - Respin Feature: Initial bonus spin is 3, and reset to 3 when new symbol appears.
  - Expand Feature: Initial bonus spin is 1, and reset to 1 when new symbol appears. The reels will change to 6x5.
  - Respin & Expand Feature: Initial bonus spin is 3, and reset to 3 when new symbol appears. The reels will change to 6x5.
