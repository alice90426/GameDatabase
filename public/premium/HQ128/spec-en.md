# Game Information

## Introduction
- Size: 3 reels, 3 rows.
- Lines: 1 Lines.
- Features: Mix Symbol, Bonus Wheel.
---

## Symbols
|  Symbol     | Description     |
|-------------|-----------------|
| [H1]~[H5]   | High Pay        |
| [L1]~[L3]   | Low Pay         |
| [SC]        | Scatter         |

### Scatter Symbol
- All reels with [SC] trigger Bonus Wheel.
- More [SC] and higher multipliers up to 3X while Bonus Wheel is triggered.
---

## Game Features

### Bonus Wheel
- Spin and get current multiplier X total play amount as award. Award is up to 900X.
