# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243 路。
- 特色：消除掉落，遞增倍數，免費遊戲。
---

## 圖標
|  圖標     | 描述 |
|-----------|------|
| [H1]~[H5] | 高賠 |
| [L1]~[L4] | 低賠 |
| [WD]      | 百搭 |
| [SC]      | 分散 |

### 百搭圖標
- [WD] 僅出現在盤面 3、4 和 5 上。
- [WD] 替代除 [SC] 之外的所有圖標。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將觸發免費遊戲和 15/20/25 免費局數。
---

## 遊戲特色

### 消除掉落
- 所有獲勝的圖標都會消除，從而使它們上方的圖標向下掉落，進行新一輪。
- 每輪都會計算額外的獲勝組合，直到無法計算出更多的獲勝組合。

### 免費遊戲
- 每一個得分都會將倍數增加一級，消除結束後倍數會重置。
- 倍數級別為：X1、X2、X3、X4、X5、X10。
