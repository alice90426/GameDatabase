# 遊戲資訊

## 簡介
- 盤面：5滾輪，4列。
- 線數：40線。
- 特色：隨機百搭、黏性百搭、圖標升級、免費遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H5] | 高賠     |
| [L1]~[L5] | 低賠     |
| [WD]      | 百搭     |
| [HW]      | 超級百搭 |
| [SC]      | 分散     |

### 百搭圖標
- [HW] 僅出現在盤面 2、3 和 4 上。
- [WD]/[HW] 替代除 [SC] 之外的所有圖標。
- 當賠付線與 [HW] 連接時，付款將有 X2 倍數。

### 分散圖標
- 當 3/4/5 [SC] 圖標出現在任何地方時，將觸髮帶有 6/8/10 獎勵輪子的免費遊戲。
- 輪子將旋轉並分別獎勵 1、2 或 3 個免費局數。
---

## 遊戲特色

### 隨機百搭
- 每個 [HW] 圖標將 1、2 或 3 個 [WD] 發送到盤面上的隨機位置。

### 免費遊戲
- 所有命中的 [HW] 圖標都會保留在盤面上，直到免費遊戲結束，每次免費旋轉時在盤面上發送 [WD] 圖標 。
- 當 [SC] 圖標命中時，它會將 [WD] 圖標添加到隨機 [HW] 中，[HW] 將再發送一個 [WD]。最大發送 [WD] 計數為 3。
