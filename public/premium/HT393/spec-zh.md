# 遊戲資訊

## 簡介
- 盤面：5滾輪，4列。
- 線數：40線。
- 特色：隨機百搭、黏性百搭、圖標升級、免費遊戲。
---

## 圖標
|  圖標     | 描述     |
|-----------|----------|
| [H1]~[H5] | 高賠     |
| [L1]~[L5] | 低賠     |
| [WD]      | 百搭     |
| [SW]      | 超級百搭 |
| [SC]      | 分散     |

### 百搭圖標
- [SW]出現在第2、3或4滾輪。
- [WD]或[SW]可替代除了[SC]以外的所有圖標。
- 當賠付線與[SW]連接時，付款將有X2倍數。

### 分散圖標
- 滾輪上任意位置出現3、4或5個[SC]，即可獲得6、8或10個獎勵滾輪。
- 獎勵滾輪會自動旋轉並分別獎勵1、2或3場免費遊戲。
---

## 遊戲特色

### 隨機百搭
- 每個[SW]圖標將1、2或3個[WD]發送到盤面上的隨機位置。

### 免費遊戲
- 所有命中的[SW]圖標都會保留在盤面上，直到免費遊戲結束，每次免費旋轉時在盤面上發送[WD]圖標 。
- 當[SC]圖標命中時，它會將[WD]圖標添加到隨機[SW]中，[SW]將再發送一個[WD]。最大發送[WD]計數為 3。
