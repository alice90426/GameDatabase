# Game Information

## Introduction
- Size: 5 reels, 4 rows.
- Lines: 40 Lines.
- Features: Random Wild, Sticky Wild, Upgrade Symbol, Free Game.
---

## Symbols
|  Symbol   | Description |
|-----------|-------------|
| [H1]~[H5] | High Pay    |
| [L1]~[L5] | Low Pay     |
| [WD]      | Wild        |
| [HW]      | Super Wild  |
| [SC]      | Scatter     |

### Wild Symbol
- [HW] appears on reels 2, 3 and 4 only.
- [WD]/[HW] substitutes for all symbols except [SC].
- When payline connect with [HW], the payment will have a X2 multiplier.

### Scatter Symbol
- When 3/4/5 [SC] symbols appear anywhere will trigger the Free Game with 6/8/10 bonus wheels.
- The wheels will spin and award 1, 2 or 3 free spins each.
---

## Game Features

### Random Wild
- Each [HW] symbol sends 1, 2, or 3 [WD] to random positions on screen.

### Free Game
- All [HW] symbols that hit remain on screen until the end of the Free Game, sending [WD] symbols on the screen every free spin.
- When a [SC] symbol hits, it adds a [WD] symbol to a random [HW], the [HW] will send one more [WD]. The maximun send [WD] counts is 3.
