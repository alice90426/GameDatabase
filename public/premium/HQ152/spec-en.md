# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Swap Reel, Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H5] | High Pay        |
| [L1]~[L5] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols except [SC].

### Scatter Symbol
- 3 [SC] anywhere on reels 2, 3 or 4 trigger 8 free spins.
---

## Game Features

### Free Game
- When Free Game is triggered with 4 or more [SC], any winning combination will multiply up to 12X.
- The same symbols except [SC] appearing on any 3 reels of 1st Reel to 4th Reel and also on 5th Reel will swap the reel without the symbol to give additional win. 
- 3 [SC] anywhere on reels 2, 3 or 4 retrigger 8 free spins.
