# 遊戲資訊

## 簡介
- 盤面：5滾輪，3列。
- 線數：243路。
- 特色：倍數百搭、延展滾輪、多選項免費遊戲。
---

## 圖標
|  圖標       |  描述        |
|-------------|--------------|
| [H1]~[H5]   | 高賠         |
| [L1]~[L5]   | 低賠         |
| [WD]        | 百搭         |
| [W1]        | 超級百搭     |
| [SC]        | 分散         |

### 百搭圖標
- [WD]/[W1]可替代除了[SC]以外的任意圖標來得到連線分數。
- [WD]只出現在主遊戲第2和4滾輪，和免費遊戲第2、3或4滾輪。
- [W1]只出現在第3滾輪。
- [H1]~[H5]與[SC]得分組合包含[W1]，該線得分將額外乘上3X、4X或5X。

### 分散圖標
- 滾輪上由最左至右連續出現3、4或5個[SC]時，可獲得免費遊戲與5、10或50倍的總押注得分。
---

## 遊戲特色

### 多選項免費遊戲
- 特色選項為五種不同延展圖標與免費遊戲場次。
- 所有延展圖標只出現在第2、3或4滾輪上，可替代[L5]得到連線分數。
- 當局部延展圖標出現在得分組合中時，該滾輪將向上延展直到完整的延展圖標出現，且延展圖標上方的圖標將同時被向上推移。
- 滾輪上由最左至右連續出現3、4或5個[SC]時，可再觸發免費遊戲與5、10或50倍的總押注得分。
