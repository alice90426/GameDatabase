# Game Information

## Introduction
- Size: 5 reels, 3 rows.
- Lines: 243 Ways.
- Features: Select Free Game.
---

## Symbols
|  Symbol   | Description     |
|-----------|-----------------|
| [H1]~[H5] | High Pay        |
| [L1]~[L6] | Low Pay         |
| [WD]      | Wild            |
| [SC]      | Scatter         |

### Wild Symbol
- [WD] substitutes for all symbols.
- [WD] appears on reels 2, 3 or 4.

### Scatter Symbol
- 3, 4 or 5 [SC] from leftmost to right on adjacent reels trigger Select Free Game, and win 5x, 10x or 50x total play amount.
---

## Game Features

### Select Free Game
- Select one of five high pay symbols, with only one the same high pay symbols during Free Game.
- 3 or more [SC] from leftmost to right on adjacent reels retrigger Select Free Game.
